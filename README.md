### 개발 기획서

### 개발 단계
 - 1차 개발 (11/29 ~ 12/5)
                          i.         사진 등록 : 사용자가 자신의 코디 사진을 업로드.
                         ii.          태그 입력 및 검색 : 코디와 관련된 태그를 입력하고, 이를 검색해 스타일 추천.
                        iii.         즐겨찾기 : 좋아하는 코디를 저장.
   - 2차 개발 (12/6 ~ 12/10)
                          i.         리뷰 작성 : 코디에 대한 의견을 남겨 사용자 간 피드백 활성화.
                         ii.          개인 프로필 관리 : 사용자 스타일 기록 및 공유.
- 3차 개발 (12/11 ~ 12/15)
                          i.         구독 기능 : 특정 사용자를 팔로우하고 업데이트 확인.
                         ii.          추천 알고리즘 : 리뷰와 태그 데이터를 결합한 개인화 추천 시스템.
- 확장 기능
                          i.         사용자 간 채팅.
                         ii.          쇼핑몰 링크 연결 (마케팅 협업)
